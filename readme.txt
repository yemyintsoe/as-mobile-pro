# step1
- run XAMPP (apache & mysql)
- go to localhost/phpmyadmin
- create database in phpMyadmin, the database name is "as_mobile_pro"
- create table in the "as_mobile_pro" database, the table columns are "id", "title", and "content"
      - id (int, auto_increment)
      - title (varchar(255))
      - content (text)
# step2
- donwload "as-mobile-pro" github repo
- paste & extract the as-mobile-pro.zip folder in xampp/htdocs root folder
- run localhost/as-mobile-pro
